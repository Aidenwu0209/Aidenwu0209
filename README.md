## 代表项目

### [CareerPilot](https://github.com/Aidenwu0209/CareerPilot)

一个面向职业规划场景的 AI 应用原型。

- 基于 `FastAPI + Vue 3 + TypeScript + Docker Compose`
- 包含简历解析、岗位知识管理、人岗匹配、学习路径规划和报告导出等模块

面向 Claude Code 的 OCR skills 工具集。

- 包含 text recognition 和 document parsing 两类能力

一个基于 PP-OCRv5 的早期 OCR skill 项目。

- 是我在 OCR 工具链方向的一次实践
